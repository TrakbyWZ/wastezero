# Static assets

This folder holds images and other static assets used by the app (e.g. logos, icons). Next.js serves files under `public/` at the site root, so `public/assets/foo.png` is available as `/assets/foo.png`.

- **waste-zero-logo.png** – Login screen logo. To add or update: place the file here (or copy from `pics/`). Reference in the app as `/assets/filename.ext`.
